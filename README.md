This is a simple, responsive website built with HTML and CSS to advertise a catering service. It showcases menu options, services, contact information, and customer testimonials to attract potential clients for events like weddings, corporate gatherings, and parties.

Features
Responsive Design: Optimized for desktop, tablet, and mobile devices.
Hero Section: Eye-catching banner with call-to-action.
Menu Showcase: Displays sample dishes with images and descriptions.
Services Overview: Highlights catering packages and customization options.
Testimonials: Customer reviews to build trust.
Contact Form: Allows visitors to inquire about services.
Footer: Includes social media links and business details.

Technologies Used
HTML5
CSS3 (with Flexbox and Grid for layout)
